All user-facing text (templates, labels, messages, placeholder data) must be in Spanish.

**Why:** The project is a Spanish wedding website ("boda" = wedding). The user explicitly corrected this after the first draft was in English.

**How to apply:** Any time new content, labels, button text, flash messages, or placeholder data is added to this project, write it in Spanish.
